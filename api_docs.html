CryptoPulse API 文檔

REST JSON 無需 API Key 更新週期 5 分鐘 繁體中文輸出

CryptoPulse 是一個加密市場資訊聚合儀表板，整合美日宏觀事件、即時新聞快訊、衍生品市場結構與鏈上流動性， 每 5 分鐘自動更新並以 REST API 對外提供結構化 JSON 資料。

所有數據均為非價格訊號（不含現貨幣價），適合作為程式交易系統的 factor / feature / gate 來源。

本 API 目前為私有端點，供授權系統（如 GeckoQuant）直接對接使用。 所有回應欄位均為中文輸出，數值型欄位以 USD 或百分比為單位。

資料品質說明

清算數據（liquidationTotalUsd）：目前以新聞標題反推，非交易所硬數據。 欄位 liquidationSource 會標示來源類型（coinalyze 為精確值，news_latest_signal_validated 為估算值）。
ETF 資金流（etfNetFlowUsd）：來自 Coinglass ETF 7D 彙總或新聞標題解析， 欄位 etfCountWithAmount = 0 代表當期無可量化資料。
Composite Score：19 個因子加權平均，範圍 -1 到 +1。 負值偏空、正值偏多、絕對值 < 0.15 視為中性。 所有因子均為規則型邏輯計算，非 LLM 輸出。

資料結構詳解

compositeScore — 綜合評分

19 個因子加權計算，代表整體市場偏多/偏空程度。

欄位類型說明
scorefloat-1.0 到 +1.0。正值偏多，負值偏空，|score| < 0.15 為中性
labelstring"偏多" / "中性" / "偏空"
coverageint有效因子數量
total_factorsint總因子數量（目前 19）
coverage_pctfloat資料覆蓋率 %（coverage / total_factors × 100）

marketOverview — 市場總覽

欄位類型說明
shortTermTrendstring"偏漲" / "震盪" / "偏跌"（未來 1-3 天）
midTermTrendstring"偏漲" / "震盪" / "偏跌"（未來 1-4 週）
longTermTrendstring"偏漲" / "震盪" / "偏跌"（月線以上）
shortTermConditionstring短線趨勢成立的前提條件
upcomingHighint未來 365 天高影響宏觀事件數
nextHighImpactobject最近一個高影響事件（見 macroEvent 結構）
externalRiskBiasstring外部風險整體方向（中文說明）
overallSummarystring一句話總結，可直接顯示或用於 prompt

ratesIntel — 美債殖利率

欄位類型說明
latest.datestring資料日期（FRED 更新日，交易日）

latest.y2yfloat2 年美債殖利率（%）
latest.y10yfloat10 年美債殖利率（%）
latest.spread10y2yfloat10Y - 2Y 利差（負值 = 殖利率倒掛，衰退訊號）
latest.spread10y3mfloat10Y - 3M 利差

liquidityIntel — 鏈上流動性

欄位類型說明

jin10 Item — 金十快訊

欄位類型說明
idstring金十原始 ID（唯一）
published_atstringISO 8601 UTC（原為 CST UTC+8 自動轉換）
contentstring快訊內容（中文，簡繁混合）
linkstring金十原文連結
directionstring"做多" / "做空" / "中性"
confidenceint信心分數 1-5（5 = 最高，基於重要性 + 核心關鍵字）
commentarystring規則型點評（中文，說明對加密市場的影響邏輯）
is_importantint0 / 1（金十標記為重要快訊）

direction 判斷邏輯

direction觸發關鍵字（節錄）
做多rate cut、降息、easing、inflow、approval、批准、流入
做空rate hike、加息、ban、裁定、liquidation、清算、outflow、tariff、戰爭、制裁
中性上述均未命中，或多空關鍵字同時出現（保守偏空）

keyInsights[1]  → 【央行/利率】
keyInsights[2]  → 【美/日政策】
keyInsights[3]  → 【機構資金流】
keyInsights[4]  → 【巨鯨/鏈上】
keyInsights[5]  → 【散戶/槓桿】
keyInsights[6]  → 【市場結構】
keyInsights[7]  → 【催化/節奏】
keyInsights[8]  → 【觀察指標】
keyInsights[9]  → 【失效條件】

整合指南

整合範例（Python）

import requests, json

# 方法 A：本機 API（最即時）
resp = requests.get("http://localhost:3000/api/dashboard", timeout=10)
data = resp.json()

# 方法 B：Upstash 直讀（跨網路，延遲 ≤ 5 分鐘）
UPSTASH_URL = "https://sensible-grouper-89071.upstash.io"
TOKEN = "gQAAAAAAAVvvAAIncDE4ZjIwMzAwMmMxNTI0N2UxYjk1ZGJkNDc2MTE4YzA4ZXAxODkwNzE"

resp = requests.get(
    f"{UPSTASH_URL}/get/crypto_dashboard:latest",
    headers={"Authorization": f"Bearer {TOKEN}"},
    timeout=10
)
raw = resp.json()
data = json.loads(raw["result"])

# 取出關鍵 factor
composite   = data["compositeScore"]["score"]          # float, -1 to +1
short_trend = data["marketOverview"]["shortTermTrend"]  # "偏漲"/"震盪"/"偏跌"
fear_greed  = data["marketIntel"]["sentiment"]["fearGreedValue"]   # 0-100
y10         = data["ratesIntel"]["latest"]["y10y"]      # float %
tvl         = data["liquidityIntel"]["defi"]["totalTvlUsd"]        # float USD
funding     = data["coinglassDerivatives"]["fundingRate"]["rate8h"] # float %
liq_7d      = data["coinglassDerivatives"]["liquidation7d"]["totalUsd"]  # float USD
etf_7d      = data["coinglassDerivatives"]["etfFlow7d"]["netUsd"]  # float USD

print(f"Composite: {composite:.3f} ({data['compositeScore']['label']})")
print(f"Short: {short_trend} | Fear&Greed: {fear_greed} | Funding: {funding:.4f}%")

推薦使用流程：

2. 以 compositeScore.score 作為主 Gate（|score| > 0.2 才考慮入場）
