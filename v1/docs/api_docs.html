CryptoPulse API 文檔

REST JSON 無需 API Key 更新週期 5 分鐘 繁體中文輸出

CryptoPulse 是一個加密市場資訊聚合儀表板，整合美日宏觀事件、即時新聞快訊、衍生品市場結構與鏈上流動性， 每 5 分鐘自動更新並以 REST API 對外提供結構化 JSON 資料。

所有數據均為非價格訊號（不含現貨幣價），適合作為程式交易系統的 factor / feature / gate 來源。

本 API 目前為私有端點，供授權系統（如 GeckoQuant）直接對接使用。 所有回應欄位均為中文輸出，數值型欄位以 USD 或百分比為單位。

Rate Limit

本機 API（localhost:3000）目前無速率限制，但建議遵守以下輪詢間隔，避免不必要的資源消耗：

端點建議間隔理由
/api/dashboard每 5 分鐘主循環更新週期，更頻繁無意義
/api/jin10每 60 秒即時 proxy，上游 API 本身約每分鐘更新
/api/jin10/history頁面載入時一次SQLite 歷史，非即時資料
/api/health每 30 秒監控用途
Upstash 直讀同樣無硬性限制，但 Upstash Free Tier 每日有請求配額，建議以 GCP VM API 為主，Upstash 為備援。

資料品質說明

清算數據（liquidationTotalUsd）：目前以新聞標題反推，非交易所硬數據。 欄位 liquidationSource 會標示來源類型（coinalyze 為精確值，news_latest_signal_validated 為估算值）。
ETF 資金流（etfNetFlowUsd）：來自 Coinglass ETF 7D 彙總或新聞標題解析， 欄位 etfCountWithAmount = 0 代表當期無可量化資料。
Composite Score：24 個因子加權平均，範圍 -1 到 +1。 負值偏空、正值偏多、絕對值 < 0.15 視為中性。 所有因子均為規則型邏輯計算，非 LLM 輸出。

錯誤格式

所有端點在失敗時均回傳標準 JSON 錯誤結構，搭配對應的 HTTP 狀態碼。

HTTP 狀態碼發生情境處理建議
200 OK正常回應—
400 Bad Request缺少必要 Query 參數或格式錯誤確認 key / days 等參數正確
404 Not Found尚無資料（update-data.mjs 從未執行過）等待首次資料更新，或確認 keep-alive 是否運行
500 Internal Server ErrorSQLite 寫入失敗查看 /api/update-log 確認最後一次更新狀態
502 Bad Gateway/api/jin10 上游 API 無回應或超時等待下次輪詢，此為暫時性錯誤
503 Service UnavailableV2 資料超過 8 小時未更新（stale）回應 header 含 X-Data-Age-Hours，確認 keep-alive 是否運行

錯誤回應範例

// HTTP 404 — 無資料

  "error": "No dashboard data available. Run the update script first."
}

// HTTP 502 — jin10 上游失敗
{
  "ok": false,

  "reason": "jin10 HTTP 503"

// HTTP 500 — 寫入失敗
{

資料新鮮度驗證

每筆回應包含 generatedAt（ISO 8601 UTC），用於判斷資料是否過期。

若 generatedAt 距今超過 15 分鐘，代表 keep-alive 主循環可能中斷， 資料已過期。建議發出警告或暫停使用該批資料作為交易 gate。

新鮮度驗證範例（Python）

from datetime import datetime, timezone, timedelta

def is_fresh(data, max_age_minutes=15):
    generated_at = data.get("generatedAt")
    if not generated_at:
        return False
    ts = datetime.fromisoformat(generated_at.replace("Z", "+00:00"))
    age = datetime.now(timezone.utc) - ts
    return age < timedelta(minutes=max_age_minutes)

data = fetch_dashboard()
if data and not is_fresh(data):
    print(f"⚠️ 資料已過期：generatedAt = {data['generatedAt']}")
    # 暫停入場訊號，等待下次更新

coinglassDerivatives.available = false 的處理

當 coinglassDerivatives.available = false 時，以下欄位均為 null：
fundingRate、openInterest、longShortRatio、liquidation7d、etfFlow7d

此時應 fallback 至 cryptoSignalMetrics7d 的新聞估算值： 
cg = data.get("coinglassDerivatives", {})
if cg.get("available"):
    liq_7d = cg["liquidation7d"]["totalUsd"]     # 精確值
    etf_7d = cg["etfFlow7d"]["netUsd"]           # 精確值
else:
    # fallback：新聞估算值
    metrics = data.get("cryptoSignalMetrics7d", {})
    liq_7d = metrics.get("liquidationTotalUsd")  # 估算值，可能偏低
    etf_7d = metrics.get("etfNetFlowUsd")        # 可能為 0（無新聞量化）
    source = metrics.get("liquidationSource")    # "news_latest_signal_validated"
    print(f"Coinglass 不可用，使用估算值（{source}）")

V2 交易系統 API

V2 API 專為程式交易系統設計，提供結構化的 factor vector、gate conditions 與統一進出場訊號。 所有 V2 端點均包含 X-Data-Age-Hours 回應 header，資料超過 8 小時自動回傳 503。

GET /v2/trade/signal

GET /api/v2/trade/signal 無需認證

整合 composite score、momentum、gate conditions，輸出統一進出場訊號。 這是程式交易系統最主要的對接端點。

import requests, json, time
from datetime import datetime, timezone, timedelta

UPSTASH_URL = "https://sensible-grouper-89071.upstash.io"
TOKEN = "gQAAAAAAAVvvAAIncDE4ZjIwMzAwMmMxNTI0N2UxYjk1ZGJkNDc2MTE4YzA4ZXAxODkwNzE"
LOCAL_API  = "http://34.70.39.177/api"

last_generated_at = None

def fetch_dashboard():
    """嘗試本機 API，失敗 fallback Upstash"""
    try:
        r = requests.get(f"{LOCAL_API}/dashboard", timeout=8)
        r.raise_for_status()
        return r.json()
    except Exception:
        pass
    try:
        r = requests.get(
            f"{UPSTASH_URL}/get/crypto_dashboard:latest",
            headers={"Authorization": f"Bearer {TOKEN}"}, timeout=10
        )
        raw = r.json()
        return json.loads(raw["result"])
    except Exception as e:
        print(f"[ERROR] 無法取得資料：{e}")
        return None

def is_fresh(data):
    ts_str = data.get("generatedAt", "")
    if not ts_str:
        return False
    ts = datetime.fromisoformat(ts_str.replace("Z", "+00:00"))
    return (datetime.now(timezone.utc) - ts) < timedelta(minutes=STALE_MIN)

def extract_factors(data):

        "yield_spread"     : data["ratesIntel"]["latest"]["spread10y2y"],
        "stablecoin_mcap"  : data["liquidityIntel"]["stablecoins"]["totalMcapUsd"],
        "defi_tvl"         : data["liquidityIntel"]["defi"]["totalTvlUsd"],
        "whale_trend"      : data["whaleTrend"]["trend"],
        "high_risk_signals": data["marketOverview"]["highRiskSignals"],
        "funding_rate_8h"  : funding,
        "open_interest_usd": oi_usd,
        "long_short_ratio" : ls_ratio,
        "liquidation_7d"   : liq_7d,
        "etf_flow_7d"      : etf_7d,
        "cg_available"     : cg.get("available", False),

def on_new_data(factors):
    """資料更新時的處理邏輯（接入 GeckoQuant 的地方）"""
    print(f"[{factors['generated_at']}] "
          f"composite={factors['composite_score']:.3f}({factors['composite_label']}) "
          f"short={factors['short_trend']} "
          f"F&G={factors['fear_greed']} "
          f"funding={factors['funding_rate_8h']}")
    # TODO: 傳入 GeckoQuant factor pipeline

# ── 主輪詢迴圈 ──
while True:
    data = fetch_dashboard()
    if data:
        if not is_fresh(data):
            print(f"⚠️ 資料過期（generatedAt={data.get('generatedAt')}），跳過本輪")

回應格式

import requests, json

UPSTASH_URL = "https://sensible-grouper-89071.upstash.io"

    f"{UPSTASH_URL}/lrange/jin10%3Ahistory/0/49",
    headers={"Authorization": f"Bearer {TOKEN}"}, timeout=10
)
raw_list = r.json()["result"]
items = [json.loads(x) for x in raw_list]

# 計算最近 50 筆方向分布
bull    = sum(1 for i in items if i["direction"] == "做多")
bear    = sum(1 for i in items if i["direction"] == "做空")
neutral = sum(1 for i in items if i["direction"] == "中性")
bias    = (bull - bear) / max(len(items), 1)  # 正值偏多，負值偏空

print(f"jin10 sentiment: 做多={bull} 做空={bear} 中性={neutral} bias={bias:.2f}")
